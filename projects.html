Classical Shadows — Quantum State Estimation

A NumPy implementation of the random-Pauli classical-shadows protocol (Huang–Kueng–Preskill, 2020) on a from-scratch statevector simulator. Estimates energies, observables, and state fidelity from few measurements — with median-of-means estimators, a test suite, and sample-complexity scaling experiments.

Nuclear · Code

Reactor Kinetics — Transient Analysis Toolkit

A point reactor kinetics solver coupling validated delayed-neutron data from PyRK with a stiff SciPy integrator, temperature feedback, and closed-form physics checks. Reproduces delayed-critical periods, prompt excursions, control-rod scram, and a self-limiting reactivity accident — validated against the inhour equation to 1e−4.

Charge-to-Mass Ratio for the Electron

PHY294 lab report. Measured e/m by observing circular electron-beam trajectories in Helmholtz-coil magnetic fields. Linearized fits at constant voltage and constant current yielded e/m = (1.7 ± 0.2)×10¹¹ C/kg, consistent with the accepted value.
